# Personal Portfolio Website

Welcome to the repository for my personal portfolio website! This project serves as a digital hub to showcase my skills, projects, and experiences as a software developer and student.

---

## ✨ Features
* **Dynamic UI/UX:** A clean, responsive design optimized for both desktop and mobile viewing.
* **Project Showcase:** Dedicated section highlighting my key projects, including an **AI Jarvis Assistant** and a **Student Grade Management System**.
* **Tech Stack Display:** A visual breakdown of the programming languages and tools I work with daily.
* **Contact Form:** A direct way for recruiters and collaborators to get in touch with me.

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **Backend (Optional):** Python
* **Version Control:** Git & GitHub
